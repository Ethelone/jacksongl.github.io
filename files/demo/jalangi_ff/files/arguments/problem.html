Changes to the "arguments" Object

The arguments object is an array-like built-in object corresponding to the arguments passed to a function. Programmer can refer to a function's arguments within the function by using the arguments object. This object contains an entry for each argument passed to the function. E.g., arguments[0] refers to function argument a in function f. Although the use of arguments provides flexibility in JavaScript programming, unncessary use of this dynamic feature adds more uncertainty to the code which is harder to be analyzed by compiler. There are numerous ways to use arguments that causes the function to be de-optimized.
