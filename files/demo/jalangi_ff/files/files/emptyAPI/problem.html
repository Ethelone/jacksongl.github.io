An Empty Analysis Template

This is an empty analysis template, containing all APIs of Jalangi. Try adding statements in those empty functions (hooks) to see what you can do with Jalangi. Note that there are two set of APIs of Jalangi, Analysis and Analysis2. Those APIs are almost the same, except a few differences:
1. Analysis supports record and replay on Node.js.
2. Analysis2 is cleaner: In Analysis2, return a value only if you need to change the operation's result, while in Analysis returning a value is mandatory.
That means Analysis has more capabilities, while Analysis2 is cleaner and more efficient.

All examples in this demo page use the first set of API. Mainly because this demo page was build before Analysis2 was created. Knowing one set of API, you can easily transfer to another API without much effort, since they are almost the same.

You can also use this demo page to quickly prototype and test an dynamic analysis in your mind, especially when you are unsure about its feasibility.
